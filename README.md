## Portfolio Projects

### 1. Guru99 Bank — API Automation & Manual Regression
A comprehensive testing project designed to keep tight traceability between manual test workbooks and automated API scripts for a simulation banking platform.

*   **Key Highlights:**
    *   **Postman Collection Strategy:** Built a modularized collection folder structure matching specific application modules *(Authentication, Deposits, Withdrawals, Transfers)*.
    *   **Newman CLI Execution:** Configured the collection to run automated regression cycles from the command line, generating environment variables (`baseUrl`, `customerId`) dynamically across chained requests.
    *   **Data-Driven Tests:** Integrated `.csv` and `.json` data sets for automated negative and boundary testing *(e.g., verifying system rejection of negative deposits)*.

### 2. Parabank — Master Test Plan & Traceability Matrix
An end-to-end master test plan outlining the structured strategy needed to validate complex financial workflows in a virtual banking environment.

*   **Key Highlights:**
    *   **Requirement Traceability Matrix (RTM):** Mapped functional business rules *(Registration, Loan Requests, Bill Payments)* directly to specific test outcomes to ensure 100% test coverage.
    *   **Defect Identification:** Isolated edge cases within multi-step transaction limits, simulating functional constraints and visual error-handling validations.

---

### 3. SauceDemo — E-Commerce Functional Testing
A client-side test suite executing deep UI, layout, and functional flow verification across an interactive e-commerce sandbox.

*   **Key Highlights:**
    *   **E2E Workflow Validation:** Tested state persistence in shopping carts, product catalog sorting algorithms *(A-Z, Price High/Low)*, and multi-step secure checkout forms.
    *   **Cross-Browser Testing Matrix:** Formulated guidelines checking UI stability and rendering parity across Google Chrome and Microsoft Edge.
